# Results

The results folder with its subfolders corresponds to the place where the various results are stored.

```
results/
    ├── data                                (contains the data saved in json files)
    │   ├── network_stats                   (contains the mean properties of the different populations (-get-network-stats))
    │   ├── phase_diag                      (contains the data generated for the phase_diag (-phase-diag))
    │   ├── sensitivity_analysis            (contains the data generated with --sensitivity-analysis)
    │   ├── sensitivity_analysis_link       (contains the data generated with --sensitivity-analysis-links)
    │   └── sim_mart_vaq                    (contains the data (ratio of agent types & topological measurements with -sim-mart-vaq)
    ├── figures                             (contains the figures generated by the [Plotter](../utils/plotter.py))
    ├── notebooks                           (contains a notebook for presentation)
    └── video                               (contains the gif to visualise the simulation)
```
