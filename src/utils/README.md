# Utils

In this folder, various utility functions are contained.

```
├── animation.py                  (functions to create the animation)
├── logger.py                     (functions to set the logger)
├── mt19937ar.c                   (functions in C to speed up random generator)
├── mt19937ar.h                   (functions in C to speed up random generator)
├── mt.pxd                        (functions in C to speed up random generator)
├── mt_random.c                   (functions in C to speed up random generator)
├── mt_random.pyx                 (functions in C to speed up random generator)
├── numpy_array_encoder.py        (function to encode numpy arrays to json)
├── plotter.py                    (function to plot the results)
├── sensitivity_analysis.py       (functions to run the sensitivity analysis)
├── stats.py                      (functions to perform statistical tests on the results)
├── suppress_stdout_stderr.py     (function to suppress deep warnings)
└── tools.py                      (function to take care of directories)
```
